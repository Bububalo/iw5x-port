# IW5xport

Open source IW5 asset exporter/converter based on Open-IW5 and [IW4 Open Formats](https://github.com/iw4x/iw4-open-formats), targeting the [IW4x ZoneBuilder](https://github.com/iw4x/iw4x-client/).
